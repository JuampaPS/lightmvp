# Galería de Imágenes - DDS Experiences

Esta carpeta contiene las imágenes utilizadas en la sección de galería de la landing page.

## Estructura de archivos

```
gallery/
├── 01-outdoor-festival.jpg
├── 02-club-night.jpg
├── 03-dj-setup.jpg
├── 04-concert-stage.jpg
├── 05-immersive-show.jpg
└── 06-urban-event.jpg
```

## Especificaciones recomendadas

- **Formato**: JPG, PNG o WebP
- **Resolución**: Mínimo 1200x800px
- **Tamaño**: Máximo 2MB por imagen
- **Aspecto**: 3:2 o 4:3 para mejor visualización en grid

## Cómo actualizar las imágenes

1. Reemplaza las imágenes existentes con tus propias fotos
2. Mantén los nombres de archivo descriptivos
3. Actualiza las referencias en `components/LightshowAudioLanding.tsx`
4. Asegúrate de que las imágenes estén optimizadas para web

## Tipos de eventos recomendados

- Festivales al aire libre
- Clubes nocturnos
- Conciertos y shows
- Eventos corporativos
- Bodas y celebraciones privadas
- Eventos urbanos y culturales
